Python vs Scratch

Similarities

Python and Scratch both use variables for labelling.

Differences

In Python, you have to type everything, while in Scratch, you can drag blocks. Also, in Python, the variables have to be named a certain way, otherwise there will be an error or it will not be stylish; however, in Scratch, you can name your variables anything you want to.
